In January, I applied for some funding from my department. With this application, came the need to update my resume, or CV as we call them here in Norway. While I was somewhat satisfied with my CV in word (ghasp!), I wanted to be able to use markdown. Specifically rmarkdown. I knew there had to be a way to make a good CV that would update nicely with new information.

I did some searching and found the [vitae]{https://github.com/mitchelloharawild/vitae} package, and thought that looked cool. The package has a good website to get examples and information about it from. If you decide on making your CV with it, I really recommend having a look.

First things first, downloading the package

remotes::install_github("mitchelloharawild/vitae")

secondly, since vitae requires latex to work, you will likely need to get that installed and working too: {r. eval = FALSE} install.packages('tinytex') tinytex::install_tinytex()
